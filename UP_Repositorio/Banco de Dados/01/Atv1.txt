1- Banco de dados é um sistema estruturado para gerenciar informações.
Eles trabalham normalmente com tabelas, onde existem campos e registros.
Nelas os dados são cruzados para criarem informação e assim gerar conhecimento.
Existem os tipos relacionais, onde a estrutura do banco de dados é definida e estruturada na sua concepção.
E os não relacionais, que não possuem uma estruturação prévia.

2- Os dados é a forma mais básica de um fato, seja ele um número, cadeia de caracteres ou até um estado.
A informação é gerada quando os dados se cruzam, assim se correlacionando entre si.
O conhecimento surge da interpretação da informação de uma forma a se tomar uma ação.

3- Um sistema de arquivos trabalha com uma estrutura de gerenciamento de dados hierarquica, onde cada arquivo é tratado como independente.
O SGBD é um sistema baseado em dados em tabelas, e essas tabelas se relacionam com outras tabelas.
Isso trás um estrutura muito mais complexa, onde relações e restrições podem ser definiidas.

4- Aqueles que usam o banco de dados são as pessoas que vão precisar das informações deles para gerar conhecimento para realizar ações inteligentes.

5- As redes sociais como facebook e instagram utilizam esse sistema de bigdata complexa para analisar os gostos dos úsuarios para trazerem anuncios relativos ao que coletaram.
